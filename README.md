# WaterFire-game

## 프로젝트 소개
네트워크 2인용 물/불 게임

## 프로젝트 기능

* 물,불로 2인용 게임
* 로비에서 게임 테마를 선택할 수 있다.
* 물은 파란색, 불은 빨간색 보석만 먹을 수 있고 모든 보석을 다 먹고 최종 위치까지 도착하면 성공
* 물이 불 구덩이에 들어가면 처음 위치, 불이 물 구덩이 들어가면 처음 위치로 이동
* 조작키로는 좌/우 이동, 업키는 점프, 스페이스바는 더블 점프
* 제한 시간 1분으로 하여 시간 내에 최종 위치에 도착해야 성공, 시간 내에 도착하지 못하면 실패
* 서로 채팅 가능
